✅ Live API — subscriber count fetched automatically

Substack Newsletter

📡 Posts via RSS · Subscribers: manual entry
  1. Your RSS feed is at yourname.substack.com/feed
  2. Recent posts appear automatically in the Content tab
  3. Substack has no public API for subscriber counts
  4. Find your count: Substack Dashboard → Settings → Subscribers

Google Analytics

📊 Via Google Sheets — free, no server needed
  1. Open Google Sheets → new spreadsheet
  2. Extensions → Add-ons → Get add-ons → search Google Analytics → Install
  3. Extensions → Google Analytics → Create new report
  4. Pick metrics: ga:sessions, ga:users, ga:pageviews — last 6 weeks
  5. Run the report, then: File → Share → Publish to web
  6. Select your sheet tab → format: CSV → click Publish
  7. Copy the published URL and paste it below
