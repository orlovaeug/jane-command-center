Google Analytics

📊 Via Google Sheets — free, no server needed
  1. Open Google Sheets → new spreadsheet
  2. Extensions → Add-ons → Get add-ons → search Google Analytics → Install
  3. Extensions → Google Analytics → Create new report
  4. Metrics: ga:sessions, ga:users, ga:pageviews · Dimensions: ga:date · last 42 days
  5. Run report → File → Share → Publish to web → CSV → Publish
  6. Paste the published URL below

For audience geo/device data: create a second report with dimensions ga:country or ga:deviceCategory and publish it separately.
